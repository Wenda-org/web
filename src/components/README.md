# Components

This directory contains reusable React components.

## Structure

- Create component folders with the component name
- Each component folder should contain:
  - `ComponentName.tsx` - The component file
  - `ComponentName.test.tsx` - Component tests (optional)
  - `index.ts` - Export file

## Example

```
components/
  ├── Button/
  │   ├── Button.tsx
  │   ├── Button.test.tsx
  │   └── index.ts
  └── Card/
      ├── Card.tsx
      └── index.ts
```
